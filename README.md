# GeoFS-Alarm
An extension to GeoFS to alarm you so that you know what is going on.

Current Version: 0.0.7

Alarms: Stall, Bank Angle, Terrain, Sinkrate, Altitude Callouts, Retard, Too Low Flaps, Too Low Gear, AP Disconnect, Minimum, & Don't Sink.

Version 0.0.8

Release: 2/20/2026

Updates: Seperating callouts between Airbus & Boeing models

Unlike other people's alarm system addons for GeoFS, this one requires no installation of TamperMonkey.
