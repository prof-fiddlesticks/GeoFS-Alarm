# GeoFS-Alarm
An extension to GeoFS to alarm you so that you know what is going on.

Current Version: 0.0.5 Alpha 3

Alarms: Stall, Bank Angle, Terrain, Sinkrate, Altitude Callouts, Retard

Version 0.0.6

Release: 2/5/2026

Updates: Adding Don't Sink, flaps, & gear

Version 0.0.7

Release: 2/6/2026

Updates: Adding minimum, & ap disconnect 

Version 0.0.8

Release: 2/8/2026

Updates: Seperating callouts between airbus and boeing models

Unlike other people's alarm system addons for GeoFS, this one requires no installation of TamperMonkey.
